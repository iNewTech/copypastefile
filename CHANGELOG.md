# Change Log

All notable changes to the "Copy Paste File" extension will be documented in this file.

## [3.1.0] - 2026-03-12

### Added
- Append-to-clipboard support for Copy Files for AI Context — selecting additional files appends to the existing AI context block instead of replacing it

### Improved
- File tree rendering and summary messages for AI context commands

## [3.0.0]

### Added
- **Copy Files for AI Context** — select files and folders to copy in AI-ready markdown format with file tree and syntax-highlighted code blocks
- **Copy Open Editors for AI Context** — copy all open editor tabs in AI-friendly format
- **Move File To...** — move files/folders to any workspace location with quick-pick browser
- **Copy File To...** — copy files/folders to any workspace location
- **Duplicate Folder** — recursively duplicate entire directory trees
- Recent destination tracking for Move/Copy File To commands
- Smart conflict resolution (overwrite, auto-rename, or cancel)
- Create new folders on-the-fly during Move/Copy operations

## [2.0.0]

### Added
- `Duplicate File` command to duplicate a file with a new name
- `Copy File Content` command to copy file content to the clipboard

## [1.0.0]

### Added
- Initial release of Copy Paste File
